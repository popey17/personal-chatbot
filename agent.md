# Agent Context

This file provides context for AI agents working on the Personal Chatbot project.

## Project Overview
A personal chatbot application capable of processing documents (PDF/TXT), extracting text, and facilitating intelligent interactions.

## Tech Stack
- **Backend**: Node.js (Express)
- **Frontend**: React (Vite)
- **Processing**: Multer, PDF-Parse, Text Chunking
- **Styling**: Vanilla CSS (Modern design)

## Directory Structure
- `/backend`: Node.js server handling file uploads, text processing, and AI logic.
- `/frontend`: React application for the user interface.

## Mission
Create a premium, document-aware personal chatbot experience.

## Roadmap
- [x] Setup backend modular structure.
- [x] Implement PDF/TXT upload and text extraction.
- [x] Implement text chunking (300 words per chunk).
- [ ] Implement vector search / RAG logic.
- [ ] Build React frontend for document management and chat.

## Guidelines for Agents
- Maintain separation of concerns between frontend and backend.
- Prioritize clean, documented code and modern UI/UX design.
- Use `agent.md` or `task.md` to track long-term goals and immediate tasks.
